## [0.1.0] - 2026-03-24

### Added
- Initial release of Windows installation guide and Scoop bucket
- PowerShell 7+ installer script (`Install-MDEMG.ps1`)
- Scoop bucket integration for easy package management
- Support for Windows 10/11 with amd64 architecture
- Docker Desktop integration for Neo4j database management
- Windows Credential Manager for secure secret storage
- Tab completion for PowerShell
- Manual installation and binary download instructions
- Troubleshooting guide for common Windows issues
- Support for OpenAI and Ollama embedding providers
